# Wafer Defect Classification (Edge-AI)

## Problem
Manual wafer inspection is slow and costly.

## Solution
AI-based wafer defect classification using MobileNetV2.

## Dataset
WM-811K wafer dataset  
7 defect classes

## Model
MobileNetV2 transfer learning  
80% validation accuracy

## Deployment
Converted to ONNX for edge devices.

## Impact
• Faster inspection  
• Lower cost  
• Industry 4.0 ready
